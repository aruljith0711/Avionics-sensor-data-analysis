# Avionics-sensor-data-analysis
End-to-end Python pipeline to analyze real flight telemetry data from NASA's DASHlink dataset. Processes 63,964+ records across 3 aircraft, detects sensor anomalies using Z-score and IQR statistical methods, and visualizes subsystem health across 6 key flight parameters — altitude, airspeed, pitch, roll, engine RPM, and temperature.
